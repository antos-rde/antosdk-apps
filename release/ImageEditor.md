# Image Editor
Full featured image editor using HTML5 Canvas.

This application is a wrapper of the [TOAST UI Image editor](https://github.com/nhn/tui.image-editor).

**Features:**

* Load image to canvas
* Undo/Redo (With shortcut)
* Crop
* Flip
* Rotation
* Free drawing
* Line drawing
* Shape
* Icon
* Text
* Mask Filter
* Image Filter
